연구 개요

드론 기반 원격 모니터링은 추가적인 지상 인프라 없이 목표 지역을 효과적으로 관측할 수 있는 비용 효율적 수단으로, 재난 감시, 시설물 안전 점검, 산림 자원 관리 등 다양한 분야에서 활용되고 있습니다.

광범위한 지역을 대상으로 하는 광역 모니터링의 경우, 사전 계획된 비행 경로를 따라 드론이 획득한 이미지를 정합하여 통합 뷰를 생성하는 과정이 필수적입니다. 그러나 최근 카메라 센서의 해상도 향상으로 인해 드론이 촬영한 이미지의 용량이 급격히 증가하였고, 이는 데이터 전송 및 통합 뷰 생성의 처리 지연 증가를 야기합니다.

본 연구에서는 광역 모니터링의 지연을 초래하는 두 개 병목 구간인 데이터 전송과 이미지 정합의 성능 개선을 위한 기법을 제안합니다.

첫째, MIDTP(Massive Irregular Data Transport Protocol)를 제안합니다. MIDTP는 로컬 네트워크 환경에 최적화된 전송 프로토콜로, 패킷 손실률이 낮은 (W)LAN 환경에서 ACK 응답을 최소화하도록 설계되어 제어 오버헤드를 줄이는 동시에 신뢰성 있는 데이터 전송을 보장합니다. 실험 결과, TCP 대비 약 59% 향상된 전송 속도를 달성했습니다.

둘째, MIS(Match based Image Selection) 기법을 제안합니다. MIS는 인접 이미지 간 특징점 매칭 결과를 기반으로 중복도가 높은 이미지를 선별적으로 제거하여, 통합 뷰 생성에 필수적인 이미지만 선별적으로 전송 및 정합합니다. 이를 통해 정합 결과의 품질 저하를 최소화하면서 데이터 전송량과 정합 연산의 복잡도를 낮춥니다.

제안하는 기법의 유효성을 검증하기 위해 자체 구축한 드론 영상 데이터셋을 활용했으며, NIQE와 세그멘테이션 기반의 평가 지표를 사용하여 정합 결과의 품질을 정량적으로 분석하였습니다. 다양한 환경에서 수집된 데이터셋을 대상으로 한 성능 평가 결과, 9.2GB 규모의 산업단지 데이터셋 기준으로 약 1시간 27분이 소요되던 작업을 약 6분 14초로 단축하여 92.8%의 시간을 절감할 수 있었습니다.

본 연구는 드론 기반 광역 모니터링 시스템의 실시간성을 대폭 향상시켜, 재난 대응, 환경 감시, 정밀 농업 등 신속한 의사결정이 요구되는 분야에서의 실용적 활용 가능성을 제시합니다.

제안 기법

MIDTP: 대용량 비정형 데이터 전송 프로토콜

MIDTP(Massive Irregular Data Transport Protocol)는 저손실 지역망 환경에서 대용량 파일 전송을 위해 설계된 전송 프로토콜입니다. 기존 TCP가 모든 세그먼트마다 ACK 응답을 요구하는 것과 달리, MIDTP는 에폭(Epoch) 단위로 누적 검증을 수행하여 제어 오버헤드를 최소화합니다.

핵심 특징

  • 종료 ACK 중심 제어: 에폭의 마지막 세그먼트만 ACK 응답
  • 선택적 재전송: 누락된 세그먼트만 재전송하여 대역폭 효율 극대화
  • 경량 헤더 구조: 세그먼트당 8바이트 헤더로 최소 오버헤드
  • 타임아웃 기반 복구: 제어 신호 손실 시에도 전송 완료 보장

실험 결과, MIDTP는 세그먼트 크기 8,760바이트 환경에서 TCP 대비 약 41.7% (93.0Mbps), QUIC 대비 약 32.2% 향상된 전송 속도를 달성했습니다. 패킷 손실 환경(1-2%)에서도 TCP 대비 최대 169%, QUIC 대비 최대 248%의 성능 향상을 보이며, 100%의 전송 성공률을 유지했습니다.

MIS: 특징점 매칭 기반 이미지 선별

MIS(Match based Image Selection)는 인접 프레임 간 특징점 매칭 결과를 활용하여 중복도가 높은 이미지를 선별적으로 제거하는 기법입니다. 드론이 연속적으로 촬영한 이미지는 높은 중복도를 가지며, 이는 불필요한 데이터 전송과 정합 연산 부하를 초래합니다.

알고리즘 원리

  1. 특징점 매칭 분석: 최초 모니터링에서 이미지 간 대응점 수 계산
  2. 중복도 평가: 각 이미지가 인접 이미지들과 공유하는 특징점 총합 산출
  3. 탐욕적 선별: 매칭 수가 많은 순서대로 이미지 제거
  4. 동적 갱신: 제거 시마다 인접 이미지의 매칭 수 업데이트하여 정보 공백 방지

MIS는 데이터량을 최대 90%까지 감소시키면서도 간격 샘플링 및 Skeletal Graph 방식 대비 우수한 시각적 품질(NIQE)과 객체 인식률(IoU)을 유지합니다. 특히 Park와 Yard 데이터셋에서 각각 8.8%, 11.3%의 IoU 향상을 달성했습니다.

실험 결과

실험 환경 및 데이터셋

DJI Mavic 3 Enterprise를 이용하여 고도 약 120m에서 다양한 환경의 드론 영상 데이터셋을 구축했습니다. 총 5종의 데이터셋(Farm, Park, City, Village, Yard)을 대상으로 성능을 평가했습니다.

MIDTP 전송 성능

저손실 환경 (0% 패킷 손실)

  • 최고 전송 속도: 93.04 Mbps (세그먼트 크기 8,760 바이트)
  • TCP 대비: +41.7% (65.68 Mbps → 93.04 Mbps)
  • QUIC 대비: +32.2% (70.4 Mbps → 93.04 Mbps)
  • 전송 성공률: 100% (모든 세그먼트 크기에서)

패킷 손실 환경

  • 1% 손실: 80.56 Mbps (TCP 대비 +169%)
  • 2% 손실: 75.52 Mbps (TCP 대비 +162%)
  • UDP: 모든 손실 환경에서 전송 실패

MIS 이미지 선별 성능

처리 시간 단축

  • Farm 데이터셋: 67분 → 3분 (96% 단축, 90% 제거 시)
  • Village 데이터셋: 94% 시간 절감
  • 데이터량 감소: 최대 90% (선택 가능)

품질 평가 (90% 제거 시)

NIQE (낮을수록 좋음)
  • Yard: 6.16 vs 간격샘플링 6.49 (5.1% 개선)
  • Yard: 6.16 vs Skeletal 7.50 (17.9% 개선)
  • 모든 데이터셋에서 최고 성능
IoU (높을수록 좋음)
  • Yard: 0.687 vs 간격샘플링 0.617 (11.3% 개선)
  • Village: 0.847 vs 간격샘플링 0.794 (6.7% 개선)
  • Park: 0.678 vs 간격샘플링 0.623 (8.8% 개선)

결론 및 기여

본 연구는 드론 기반 광역 모니터링 시스템의 두 가지 주요 병목 구간인 데이터 전송과 이미지 정합을 동시에 해결하는 통합 솔루션을 제시했습니다.

주요 기여

  • MIDTP 프로토콜: 저손실 지역망 환경에 최적화된 경량 전송 프로토콜로 TCP 대비 최대 41.7% 성능 향상
  • MIS 알고리즘: 특징점 매칭 기반 이미지 선별로 데이터량 최대 90% 감소하면서도 품질 유지
  • 종단 간 최적화: 두 기법의 통합으로 전체 처리 시간 92.8% 단축 (1시간 27분 → 6분 14초)
  • 실시간 모니터링: 드론 착륙 후 현장에서 즉각적인 상황 파악과 의사결정 가능

향후 연구 방향

  • 비행 경로 최적화: MIS의 중복도 정보를 활용한 사전 비행 계획 수립
  • 적응형 MIDTP: 네트워크 상태에 따른 동적 세그먼트 크기 조절
  • 목적 지향 선별: 특정 관심 영역이나 객체 유형을 우선 보존하는 선택적 정합

본 연구의 결과는 재난 대응, 시설물 관리, 정밀 농업 등 신속한 의사결정이 요구되는 모든 드론 기반 모니터링 분야에 적용 가능하며, 실시간 광역 모니터링 시스템의 실용화에 기여할 것으로 기대됩니다.
